Frequently asked questions

How do I open a Markdown file?
Tap any .md or .markdown file in Files.app, iCloud Drive, or any file provider — MDReader will open it directly. You can also use the document browser that appears when the app launches to browse and create files.
Can I edit and preview at the same time?
Yes. Use the Split mode in the toolbar at the bottom (iOS) or top (Mac) to show the editor and the rendered preview side by side. Both panels scroll in sync.
How do Mermaid diagrams work?
Write a fenced code block with the language set to mermaid. MDReader renders it automatically using a bundled Mermaid library — no internet connection required. Tap any diagram to expand it full-screen.
Are my documents synced across devices?
MDReader stores documents in iCloud Drive by default. Any device signed into the same Apple ID with iCloud Drive enabled will have access to the same files. MDReader itself has no server — sync is handled entirely by Apple's iCloud infrastructure.
Does MDReader work on Apple Vision Pro?
Yes. MDReader runs natively on visionOS as a spatial window. You can open, edit, and preview Markdown documents in your space just as you would on iPhone or iPad.
I purchased the app — do I need to buy it again on another device?
No. MDReader is a universal purchase. Once bought, it is available on all your iPhone, iPad, Mac, and Apple Vision Pro devices linked to the same Apple ID at no extra cost.
Is there a subscription?
No. MDReader is a one-time purchase of 1.99 €. No subscriptions, no in-app purchases.
What file types does MDReader support?
MDReader opens .md and .markdown files. Plain text files (.txt) can also be opened and edited.
